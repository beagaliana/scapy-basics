#Proposed exercises with Scapy
#######################################

1. Create a Python virtual environment and install Scapy.

2. Capture a UDP packet and a TCP packet. Sniff it using Wireshark.

3. Create an script that reads a packet capture and dissects it.

4. Craft an HTTP Get request.

5. Implement your own traceroute this time measuring transit delays of packets.
